Frequently Asked Questions!

What is the Document Object Model (DOM)?

Document Object Model is like a map of a web page. A html page contains elements like headings, paragraph and images or links. This particular structure looks like tree which is called node in the language of programming. DOM Can be used interact with those node with the help of javascript to change the behavior or the style of those elements.


How do you select an element from the DOM?

There are many ways to select an element from the DOM. Most common practices are document.getElementById which select the element with ID, document.getElementByClassName which select all elements having the particular class, document.querySelector which selects the first element from the class.


What is event delegation in the context of the DOM, and why is it useful?

Event delegation in Dom is the practice where you can handle events such as click to a parent element without adding event listeners to each of the elements which eventually saves time and Event delegation can also be used to add events for new elements which you can add later to your page.


How do you manipulate an element's attributes and styles using the DOM?

We can use setAttribute(), getAttribute() & removeAttribute() to manipulate and elements attributes and we can use for example h1.style.color = 'blue'; to change the color of all the h1 elements. which works for most of the css styles. We can also achieve this with Tailwind Css as an example Element.classList.add("shadow-lg", "border-2") using this it will add class names to any element that you called using an ID
